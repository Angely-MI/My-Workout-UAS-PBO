# My-Workout-UAS-PBO
My Workout adalah aplikasi desktop berbasis Python yang digunakan untuk mencatat dan memantau aktivitas olahraga pengguna. Aplikasi ini dikembangkan sebagai tugas Ujian Akhir Semester mata kuliah Pemrograman Berorientasi Objek dengan menerapkan konsep OOP seperti encapsulation, inheritance, dan polymorphism.
