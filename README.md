# My-Workout-UAS-PBO
My Workout adalah aplikasi desktop berbasis Python yang digunakan untuk mencatat dan memantau aktivitas olahraga pengguna. Aplikasi ini dikembangkan sebagai tugas Ujian Akhir Semester mata kuliah Pemrograman Berorientasi Objek dengan menerapkan konsep OOP seperti encapsulation, inheritance, dan polymorphism.

## Fitur Aplikasi
- Input data workout (tanggal, jenis olahraga, durasi, intensitas)
- Menampilkan riwayat workout dalam bentuk tabel
- Perhitungan total dan rata-rata kalori
- Ranking olahraga berdasarkan kalori terbanyak
- Grafik batang kalori per jenis olahraga
- Filter data mingguan dan bulanan
- Penyimpanan dan pemuatan data otomatis (CSV)

## Teknologi yang Digunakan
- Python 3
- Tkinter (GUI)
- Matplotlib (Grafik)
- CSV (Penyimpanan Data)
